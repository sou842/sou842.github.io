# Deployment Steps for Vite Project to GitHub Pages

1. Make sure your project is connected to a GitHub repository and updated branch in the main
    git add .
    git commit -m "Deploying Vite project"
    git push origin main


2. Install gh-pages (if not already installed):
   npm install gh-pages --save-dev

3. In your package.json, add/modify the following scripts:

   "scripts": {
     "predeploy": "npm run build",
     "deploy": "gh-pages -d dist",
     "build": "vite build",
     "dev": "vite"
   }

4. (Optional but recommended) Add a homepage field in package.json:
   "homepage": "https://<your-username>.github.io/"

   Example:
   "homepage": "https://sou842.github.io/"

5. Build and deploy your project:
   npm run deploy

6. Wait 1–2 minutes, then visit your deployed site at:
   https://<your-username>.github.io/

# Redeploy Steps

1. Redeploy Steps (when you make new changes)
    git add .
    git commit -m "Update project"
    git push origin main

2. Save and push your latest code to main:
    npm run deploy

3. Refresh your site after 1–2 minutes to see the changes.
