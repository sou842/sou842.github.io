# Modern Developer Portfolio

A high-performance, visually stunning developer portfolio built with **Next.js 15**, **Three.js**, and **Spline**. This project features a futuristic, responsive design with advanced animations and a modular architecture.

## 🚀 Live Demo
[sou842.github.io](https://sou842.github.io/)

## ✨ Key Features
- **3D Interactive Scenes**: Integrated Spline and Three.js scenes for an immersive experience.
- **ASCII Art Integration**: Custom ASCII scenes for a unique, developer-centric aesthetic.
- **Modern UI Components**: Built with Radix UI and styled with Tailwind CSS 4.0.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.
- **Dark Mode Support**: Seamless theme switching with `next-themes`.
- **Modular Sections**:
  - Hero with Spline animations
  - Technical Infrastructure & Stack
  - Experience & Metrics
  - Testimonials & Social Proof
  - Interactive Contact & CTA

## 🛠️ Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/), [Lucide React](https://lucide.dev/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix UI)
- **3D Graphics**: [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/), [Spline](https://spline.design/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Type Safety**: [TypeScript](https://www.typescriptlang.org/)

## 🛠️ Local Development

### Prerequisites
- Node.js (Latest LTS recommended)
- npm / yarn / pnpm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sou842/sou842.github.io.git
   cd sou842.github.io
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Build & Deployment

### Build
To create an optimized production build:
```bash
npm run build
```

### Deployment to GitHub Pages
The project is configured for easy deployment to GitHub Pages using the `gh-pages` package:
```bash
npm run deploy
```
